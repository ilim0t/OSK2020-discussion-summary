```
去年余っていたし、部費減らしませんか？
```

1. 他サークルの比較すると金額に開きがある

## Active Users対策について
:::spoiler メタ情報
> [対応スレッド](https://osk2019.slack.com/archives/C010L5UGTJM/p1585280394006400)
> [time=Sun, May 31, 2020 1:17 AM]
:::

```
決議に必要な人数足りない。 出席しやすくするか、必要な人数を減らすか…
```

#### 背景
[総会の決議方法](https://github.com/TUS-OSK/OSK-REGULATION/blob/master/%E5%BF%9C%E7%94%A8%E6%95%B0%E5%AD%A6%E7%A0%94%E7%A9%B6%E9%83%A8%E8%A6%8F%E7%B4%84.md#%E7%AC%AC%E4%B9%9D%E6%9D%A1%E6%B1%BA%E8%AD%B0%E6%96%B9%E6%B3%95)として、「委任状を含め、三分の二以上部員が出席」することが前提にある。しかし、それを満たしそうにない状況にあり決議が行えないかもしれないことに困っている。
#### 意見
議論中でで出てくる人の分類として、以下のものがある。

| No  | 決議に必要 | 決議への参加意欲 | 決議への参加 |       例       |
|:---:|:----------:|:----------------:|:------------:|:--------------:|
|  (1)  |    :o:     |       :o:        |     :o:      |                |
|  (2)  |    :o:     |       :o:        |     :x:      | 現地に行けない |
|  (3)  |    :o:     |       :x:        |     :x:      |    幽霊部員    |
|  (4)  |    :x:     |        -         |      -       |     非部員     |


主な意見として以下が挙げることができる。
* (2)を(1)に変える方法 (オンライン開催等)
* (3)を(4)に変える方法 (休部制度の利用、必要条件の緩和)
  * 部費の支払いと連動して、払わない人を(4)として扱う
  * Activeかどうかで判断し、非Activeな人を(4)として扱う
  
  どちらも大多数を(4)と扱う等の不正の防止が困難、実施のコストが高いという側面がある。
  また、決議の母数から外すという行為は本来望ましくない。

* 決議に一定数の出席が必要という概念をなくす(参加義務を課す等)

#### 現状
各意見に対する議論が混ざっていて、平行線。  
Activeな人の属性を確認、現状の調査中。

## コロナ影響を受けた上での部費に関して
:::spoiler メタ情報
> [対応スレッド](https://osk2019.slack.com/archives/C010L5UGTJM/p1590469385442900)
> ==非ピン留め[^no-pin]==
> [time=Sun, May 31, 2020 10:54 AM]
:::

```
新型コロナで前期活動できないし、部費減らしたい。けれど、部費を払わないと議決権がないな…
```

#### 背景
新型コロナウイルス感染症(COVID-19)よる影響で
* 前期における活動
* 部費の回収

ができないため、部費の回収時期や金額の変更を考える必要が出てきた。
#### 意見
部則では「部費を支払ったものが部員」として認められる。  
しかし特殊状況下であるので、「2020年度活動への参加」を基準として部費を課す方針で考えている。
そこで、特殊な状況下において何を「活動」とみなすかが議論の的となっている。

Discordでの活動を
- 2019年度部員による2019年度の延長線上としての公式活動
- 2019年度部員による非公式活動(新入生とのなれあいの場)
- 2020年度部員による公式活動

のどれと捉えるかが議論になっている。

また、総会・部会は部員ベースで開催されるので、現状では限られた人しか議決権を持たないという問題がある。


#### 現状
* 前期の途中からでもオフラインでの活動が行われるならば、年4000円の徴収  
行われないのであれば年2000円の徴収とする
* 運営委員会に限っては役員として1年間を承認されているので、部費を支払っていない人でも運営委員会での議決は有効である
